# InstaClone (Flask)

A full-stack Instagram clone built using Flask, SQLAlchemy, and Bootstrap.

## 🚀 Features

- User authentication (Login/Register)
- Create, like, and comment on posts
- Follow/unfollow users
- Stories and Reels support
- Real-time messaging (Direct)
- Notifications system
- Profile management

## 🛠 Tech Stack

- Python (Flask)
- SQLite (Database)
- SQLAlchemy ORM
- Flask-Login
- HTML, CSS, Bootstrap
